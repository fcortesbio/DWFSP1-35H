¿Qué es el Document Object Model (DOM)?

El modelo de objetos de documento (DOM, por sus siglas en inglés) es una interfaz de programación para documentos web. Representa la estructura de una página web como un árbol jerárquico de objetos, lo que permite a los desarrolladores manipular el contenido, la estructura y el estilo de una página web de forma dinámica mediante lenguajes de programación como JavaScript.

El DOM representa el documento como un árbol de nodos. Cada elemento, atributo y fragmento de texto en un documento HTML es un nodo en el árbol del DOM.

Conceptos Clave del DOM

  1. Estructura de árbol:
    • El DOM representa el documento como un árbol de nodos. Cada elemento, atributo y parte de texto en un documento HTML es un nodo en el árbol DOM.
    • Por ejemplo, una estructura HTML como la siguiente: 
      <html>
    <head>
        <title>Example</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <p>Welcome to the DOM.</p>
    </body>
</html>
      Se convierte en una estructura de árbol en el DOM donde:
      • <html> es el elemento raíz.
      • <head> y <body> son elementos secundarios de <html>.
      • <title>, <h1> y <p> están anidados aún más adentro.
  2. Tipos de nodos:
    • Nodos de elementos: representan elementos de HTML (por ejemplo, <div>, <h1>, <p>).
    • Nodos de texto: Representan el contenido de texto dentro de un elemento.
    • Nodos de atributo: Representan atributos como class, id, etc.
  3. API para Interacciones:
    • Los desarrolladores pueden usar JavaScript para acceder y modificar el DOM, lo que permite realizar actualizaciones dinámicas de una página web.
    • Ejemplos de operaciones de JavaScript:
      • Seleccionar elementos: document.getElementById("id"), document.querySelector(".class").
      • Cambiar contenido: element.innerHTML = "Nuevo contenido".
      • Agregar estilos: element.style.color = "red".
      • Escuchar eventos: element.addEventListener("click", handler).
  4. Representación en vivo:
    • El DOM está activo, lo que significa que los cambios en el árbol DOM se reflejan en la página web inmediatamente.

¿Cómo funciona el sitio?

  • Entrada: La página permite a los usuarios ingresar un número en un campo de entrada.
  • Cálculo: Cuando se hace clic en el botón "Calcular", el código JavaScript verifica si el número ingresado es par o impar.
  • Salida: El resultado ("es par" o "no es par") se muestra dinámicamente en la página debajo del botón.
